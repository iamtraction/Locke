### Directory Sturcture for locales
```bash
ProjectDir/
└─── locales
     ├─── constants.yaml
     ├─── en_us
     │    ├─── error.yaml
     │    ├─── info.yaml
     │    └─── warn.yaml
     ├─── en_uk
     │    ├─── error.yaml
     │    ├─── info.yaml
     │    └─── warn.yaml
     └─── en_in.yaml
```
